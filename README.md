# Flash Reef Vault

Static crypto dashboard that turns whale movement into a confidence/fear heatmap and copy-trade cues.

## Overview

Flash Reef Vault is a real-time crypto dashboard that visualizes on-chain whale activity through an emotional temperature heatmap. It transforms whale wallet movements into actionable signals for traders who read fear as alpha.

## Features

- **Signal Heatmap**: Visual representation of whale confidence, fear, and copy-trade signals
- **Whale Tape**: Live feed of whale wallet activities with detailed notes
- **Market Mood**: Comprehensive market sentiment metrics and analysis
- **Analytics Dashboard**: Whale behavior patterns and trend analysis
- **Signal Strength**: Confidence and fear metrics with visual indicators
- **Real-time Updates**: Automatic data refresh and live bias tracking

## Run

Open `index.html` in a browser. No build step required.

## Files

- `index.html` - Main HTML structure
- `style.css` - Complete styling and responsive design
- `app.js` - Interactive JavaScript with real-time data simulation
- `vercel.json` - Vercel deployment configuration

Signals are derived from:
