# Flash Reef Vault

Static crypto dashboard concept that turns whale movement into a confidence/fear heatmap and copy-trade cues.

## Run

Open `index.html` in a browser. No build step required.

## Files

- `index.html`
- `style.css`
- `app.js`
- `vercel.json`

## Notes

- Pure static HTML, CSS, and JavaScript.
- Intended for deployment as a static site.
